## AI Document Analyser

Upload a PDF and get an AI-generated summary instantly.

### Tech Stack
- Python, Flask
- OpenAI API
- pdfplumber

### How to run
1. Install dependencies: pip install flask pdfplumber openai
2. Set your OpenAI API key: export OPENAI_API_KEY=your_key
3. Run: python app.py
4. Open http://127.0.0.1:5000
